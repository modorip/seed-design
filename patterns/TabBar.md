# TabBar

하단 5탭 네비게이션.

## 구성

- 5개 탭 고정: 홈 / 발견 / 도감 / 광장 / 나
- 각 탭: Lucide stroke 아이콘 (24px) + 라벨 (caption-2 / 600)
- 활성: `--color-primary` (#0066FF)
- 비활성: `rgba(55,56,60,0.61)`
- 배경: 흰색, 상단 1px 보더, safe-area 하단 인셋

## 인터랙션

- 탭 시 해당 탭 루트로 이동 (스택 초기화)
- 이미 활성 탭을 다시 누르면 스크롤 top
- 하위 화면에서도 해당 루트 탭이 활성 표시 유지

## 접근성

- `role="tablist"`, 각 탭 `role="tab"` + `aria-selected`
- 키보드 좌/우 방향키로 탭 이동

## 변형

- 숨김 모드: 온보딩·풀스크린 모달·발견 성공 오버레이에서 hide

## 안티 패턴

- 탭 5개를 넘기지 마세요 (모바일 인지 한계)
- 뱃지 동시 노출은 1~2개로 제한
