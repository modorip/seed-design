# EmptyState

빈 상태 표시.

## 구조

```
[일러스트 (또는 아이콘 64px)]
[제목 — title-3 / 700]
[설명 — body-2 / 500 / fg-subtle, max 2줄]
[액션 (선택)]
```

## 권장 카피

| 상황 | 카피 |
|---|---|
| 발견 자원 0 | "아직 발견한 곳이 없어요" / "주변 자원을 찾아보세요" + [발견하기 CTA] |
| 검색 결과 0 | "검색 결과가 없어요" / "다른 키워드로 시도해보세요" |
| 필터 결과 0 | "조건에 맞는 자원이 없어요" / "필터를 조정해보세요" |
| 네트워크 오류 | "잠시 후 다시 시도해주세요" / 작은 글씨로 오류 코드 |

## 사용 예

```tsx
<EmptyState
  icon={<CompassIcon />}
  title="아직 발견한 곳이 없어요"
  description="주변의 자원을 찾아 도감을 채워보세요"
  action={<Button variant="primary">발견 시작하기</Button>}
/>
```

## 안티 패턴

- 빈 상태에 농담·이모지 사용
- 액션 없는 빈 상태 (사용자가 다음 행동을 모름)
