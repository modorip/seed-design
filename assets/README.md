# 자산 (Assets)

## 폰트

| 패밀리 | 라이센스 | 출처 |
|---|---|---|
| Pretendard (9 weights, OTF) | SIL OFL 1.1 | https://github.com/orioncactus/pretendard |
| Wanted Sans Variable | SIL OFL 1.1 | https://github.com/wanteddev/wanted-sans |

Phase 0 현재는 앱 저장소 `wanted/fonts/`에 로컬 OTF 포함. Phase 1에서 패키지화 시 별도 자산 배포 검토.

## 광역 SVG

17개 광역 polygon path. 출처는 공공 행정구역 데이터 (퍼블릭 도메인). 단순화 후 별도 JSON으로 보관 예정.

## 컬러 팔레트

`tokens/컬러.md`에 시멘틱 토큰. 17광역 톤 매핑은 Phase 1에 `assets/region-palette.json` 으로 추출 예정.

## 사용 권리

본 저장소는 MIT 라이센스. 위 폰트와 광역 데이터는 각자의 원 라이센스를 따릅니다. 사용 시 출처 표기를 권장합니다.
