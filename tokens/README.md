# 토큰 (Tokens)

디자인 토큰은 디자인 결정의 최소 단위입니다. 컴포넌트는 토큰을 참조하고, 토큰은 의미(semantic)를 표현합니다.

## 토큰 명명 규약

```
--{category}-{role}-{variant?}
```

예시:
- `--color-primary` (브랜드 컬러)
- `--color-fg` (전경 텍스트)
- `--color-fg-subtle` (보조 텍스트)
- `--text-title-1-size` (타이틀 1 폰트 크기)
