# 모션

## 지속 시간 (duration)

| 토큰 | ms | 용도 |
|---|---|---|
| `--motion-instant` | 0 | 즉각 |
| `--motion-fast` | 120 | 마이크로 인터랙션 (chip, button press) |
| `--motion-normal` | 200 | 작은 상태 전환 (탭 활성, 토글) |
| `--motion-slow` | 320 | 화면 전환, 지도 줌 |
| `--motion-flip` | 700 | 3D 카드 flip |
| `--motion-reveal` | 1100 | 발견 성공 X축 720° |

## Easing

| 토큰 | 함수 | 용도 |
|---|---|---|
| `--ease-out-standard` | `cubic-bezier(.4, 0, .2, 1)` | 기본 (Wanted standard) |
| `--ease-out-strong` | `cubic-bezier(.16, 1, .3, 1)` | 강한 감속 |
| `--ease-spring` | `cubic-bezier(.34, 1.56, .64, 1)` | 발견 카드 안착 |
| `--ease-linear` | `linear` | 로딩, 스캔 라인 |

## 원칙

1. **목적 있는 모션만**: 화면 의도를 명확히 할 때만 사용. 장식 금지.
2. **사용자 진행 차단 금지**: 200ms를 넘는 전환은 백그라운드에서 계속 인터랙션 가능해야 합니다.
3. **`prefers-reduced-motion` 존중**: 운영체제 설정 시 모든 비필수 모션 비활성화.

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
