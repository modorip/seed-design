# 스페이싱

## 스케일

8px 기반 + 4px 보조.

| 토큰 | px |
|---|---|
| `--space-1` | 4 |
| `--space-2` | 8 |
| `--space-3` | 12 |
| `--space-4` | 16 |
| `--space-5` | 20 |
| `--space-6` | 24 |
| `--space-8` | 32 |
| `--space-10` | 40 |
| `--space-12` | 48 |
| `--space-16` | 64 |

## 의미 토큰

| 토큰 | 값 | 용도 |
|---|---|---|
| `--inset-page` | `var(--space-4)` | 화면 좌우 인셋 (16) |
| `--gap-section` | `var(--space-8)` | 섹션 사이 (32) |
| `--gap-card` | `var(--space-4)` | 카드 내부 패딩 (16) |
| `--gap-list` | `var(--space-3)` | 리스트 행 사이 (12) |

## 그리드

- 모바일 카드 그리드: 2열, gap 12, 좌우 인셋 16
- 칩 영역: 가로 스크롤, gap 8, 좌우 인셋 16
