# 컬러

## 브랜드 컬러

| 토큰 | 값 | 용도 |
|---|---|---|
| `--color-primary` | `#0066FF` | 브랜드 프라이머리, CTA |
| `--color-primary-hover` | `#0052CC` | hover state |
| `--color-primary-active` | `#003D99` | active state |

## 시멘틱 (라이트 모드)

| 토큰 | 값 | 용도 |
|---|---|---|
| `--color-bg` | `#F4F6F8` | 캔버스 배경 |
| `--color-surface` | `#FFFFFF` | 카드 표면 |
| `--color-fg` | `rgba(46,47,51,0.88)` | 본문 텍스트 |
| `--color-fg-subtle` | `rgba(55,56,60,0.61)` | 보조 텍스트 |
| `--color-fg-muted` | `rgba(112,115,124,0.45)` | 더 옅은 보조 |
| `--color-border` | `#E5E8EB` | 카드·입력 경계 |
| `--color-border-strong` | `#D1D6DB` | 강조 경계 |

## 상태 컬러

|---|---|---|
| `--color-success` | `#00BF40` | 성공, 발견 완료 |
| `--color-warning` | `#FF9200` | 경고, 보통 혼잡 |
| `--color-danger` | `#FF4242` | 위험, 매우 혼잡 |
| `--color-info` | `#00AEFF` | 정보 |

## 4계열 (도메인 컬러)

모두립 앱 고유 — 도감 1차 분류에 사용합니다.

| 계열 | 토큰 | HEX |
|---|---|---|
| 자연 | `--color-nature` | `#009632` |
| 유적 | `--color-heritage` | `#D17600` |
| 문화 | `--color-culture` | `#6541F2` |
| 축제 | `--color-festival` | `#D331B8` |

## 17광역 톤

각 광역에 1개씩 배정된 톤. `--color-region-{id}` 형태. 실제 값은 `assets/region-palette.md` 참조 (Phase 1).

## 다크 모드

토큰 정의는 라이트 모드 기준. 다크 모드는 `[data-theme="dark"]` 선택자로 같은 토큰을 재정의합니다. Phase 1에서 활성화 예정.

## 카드 그림자 (컴포지트)

```css
--shadow-card-1: 0 1px 2px rgba(0,0,0,.04), 0 0 0 1px rgba(0,0,0,.04);
--shadow-card-2: 0 2px 8px rgba(0,0,0,.06), 0 0 0 1px rgba(0,0,0,.04);
--shadow-card-3: 0 8px 24px rgba(0,0,0,.08), 0 0 0 1px rgba(0,0,0,.04);
```

블러는 사용하지 않습니다. 다중 박스 섀도우로 깊이를 만드는 것이 원칙.
