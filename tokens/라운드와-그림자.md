# 라운드와 그림자

## 라운드 (Radius)

| 토큰 | 값 | 용도 |
|---|---|---|
| `--radius-xs` | 4 | 칩, 작은 뱃지 |
| `--radius-sm` | 8 | 입력 필드, 작은 버튼 |
| `--radius-md` | 12 | 일반 버튼, 작은 카드 |
| `--radius-lg` | 16 | 카드 |
| `--radius-xl` | 20 | 큰 카드 |
| `--radius-2xl` | 24 | 모달, 바텀시트 |
| `--radius-3xl` | 32 | 큰 모달 |
| `--radius-full` | 9999 | pill, 아바타 |

## 그림자

블러 효과는 사용하지 않습니다. 다중 박스 섀도우로 깊이 표현.

```css
--shadow-card-1: 0 1px 2px rgba(0,0,0,.04), 0 0 0 1px rgba(0,0,0,.04);
--shadow-card-2: 0 2px 8px rgba(0,0,0,.06), 0 0 0 1px rgba(0,0,0,.04);
--shadow-card-3: 0 8px 24px rgba(0,0,0,.08), 0 0 0 1px rgba(0,0,0,.04);
--shadow-modal:  0 24px 48px rgba(0,0,0,.18), 0 0 0 1px rgba(0,0,0,.04);
```

## 적용 가이드

| 컴포넌트 | radius | shadow |
|---|---|---|
| 버튼 | `--radius-md` ~ `--radius-lg` | 없음 |
| 칩 | `--radius-xs` | 없음 |
| 카드 (기본) | `--radius-lg` | `--shadow-card-1` |
| 카드 (강조) | `--radius-lg` | `--shadow-card-2` |
| 모달 / 바텀시트 | `--radius-2xl` (상단만) | `--shadow-modal` |
| 3D 카드 모달 | `--radius-lg` | `--shadow-card-3` |
